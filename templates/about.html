Pull apart any song.
Isolate the vocals, guitar, bass — hear what's underneath. Understand what makes it work.
The Pipeline
1
Find
Search Spotify or upload your own audio file.
2
Isolate
Separate every instrument and vocal into individual stems.
3
Understand
See key, tempo, chords, and harmonic structure by section.
4
Interact

5

Technology
Demucs
Stem Separation
Meta's state-of-the-art transformer model for audio source separation, running on cloud GPU via Replicate. Isolates up to 6 stems — vocals, drums, bass, guitar, piano, and other instruments — with high fidelity. Stereo field analysis further refines each stem by panning position, producing detailed sub-stem components.
Basic Pitch
Note & Pitch Detection
Spotify's neural network for polyphonic pitch detection. Converts each isolated stem to MIDI-grade note events with accurate onset and duration. Per-instrument frequency ranges and confidence thresholds are applied to filter noise and improve accuracy.
Essentia
Key + Tempo Detection
Music analysis library for key estimation and BPM detection, applied directly to the audio signal. Results feed into the harmonic analysis pipeline and are displayed in real time during instant analysis — before stems have finished separating.
Web Audio API
Browser Mixing Engine
Browser-native audio processing powers the full mixer — mute, solo, per-stem volume, loop, and real-time pitch transposition — with no server roundtrips. Stems load in parallel and join the mix progressively. Initial fader levels are set proportionally to each stem's RMS energy, so the mix is balanced from the first play.
Diatonic Template Matching
Chord & Harmonic Analysis
Windowed pitch-class histogram analysis matched against diatonic chord templates, segmented by song section. Produces roman numeral chord progressions relative to the detected key and powers the section-aligned harmonic timeline.
Claude API
Musical Insight
Anthropic's language model generates structured musical analysis — progression names, key context, and theory-based song recommendations — from detected chords, key, tempo, and lyrics. Output is constrained to strict JSON for reliable downstream rendering.
Multi-source Audio Acquisition
Audio Pipeline
A hardened acquisition pipeline sources audio through YouTube (via proxied yt-dlp with bot detection bypass), Spotify preview, and iTunes preview — with automatic fallback at each stage. User file upload is always available as a direct path. Every failure surfaces a clear next step instead of a broken state.
Concurrent Job Queue
Multi-user Processing
Multiple users can run full analyses simultaneously. A FIFO job queue handles overflow gracefully — users waiting for a slot see live queue status and are automatically promoted when processing capacity opens, with no manual retry required.
Flask + Vanilla JS
Application Stack
Lightweight Python backend with a zero-dependency frontend. SQLite for track caching, filesystem-backed job state, and an in-memory job registry with automatic pruning. Deployed on Render with Gunicorn.
