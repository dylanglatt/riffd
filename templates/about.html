See how songs work.
Riffd uses AI audio analysis to break songs into individual parts, detect musical properties, and give you the tools to learn and play along.
The Pipeline
1
Start
Search for a song or upload audio.
2
Isolate
Separate vocals, instruments, and parts.
3
Understand
See key, chords, and structure.
4
Interact
Loop, focus, and explore sections.
5
Discover
Find songs with similar musical structure.
Technology
Demucs
Stem Separation
Meta's state-of-the-art transformer model for source separation. Isolates vocals, drums, bass, and other instruments with high fidelity.
Basic Pitch
Note Detection
Spotify's neural network for pitch detection. Converts audio to MIDI with accurate note onset and duration.
Krumhansl-Schmuckler
Key Detection
Pitch-class profiling algorithm that determines the tonal center and mode of a piece.
Web Audio API
Playback Engine
Browser-native audio processing. Enables real-time mixing, looping, and transposition without server roundtrips.
Diatonic Template Matching
Chord Analysis
Windowed pitch-class histogram analysis matched against diatonic chord templates.
Claude API
Musical Insight
Anthropic's language model generates plain-language musical analysis from detected chords, key, tempo, and structure.
Smart Recommendations
Song Discovery
AI-driven recommendations that match songs by harmonic DNA — same progressions, key relationships, and voice leading patterns.
Flask + Vanilla JS
Application Stack
Lightweight Python backend with zero-dependency frontend. Fast, simple, and maintainable.
Built for musicians
Every feature is designed for people who play instruments. Not a tech demo — a tool you actually use in your practice room.
Only what matters
We cut through the noise and show you exactly what you need to play the song.
Get better, faster
Riffd helps you learn faster, practice smarter, and improve.
Discover through theory
Find songs connected by how they sound harmonically, not by what an algorithm thinks you'll click on.
