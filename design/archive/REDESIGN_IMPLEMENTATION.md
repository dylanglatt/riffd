## Design Language Reference

These are the core visual values found in the Figma reference. Use the actual files as the source of truth — this is a cheat sheet.

### Colors

| Token | Value | Usage |
|-------|-------|-------|
| Background | `#0B0B0B` to `#0D0D0D` | Flat near-black. NO colored gradient washes or ambient glows. |
| Primary text | `#F5F5F5` | Headlines, primary content |
| Secondary text | `rgba(245,245,245, 0.55)` | Body text, descriptions |
| Muted text | `rgba(245,245,245, 0.35)` | Captions, labels, badges |
| Label text | `rgba(245,245,245, 0.4)` | Section headers, uppercase labels |
| Accent | `#D4691F` | Warm amber/orange — icons, slider thumbs, active tab underlines, monospace formula text ONLY |
| Dimmed accent | `#8A4513` | Subtle accent variant |
| Borders (standard) | `rgba(255,255,255, 0.08)` | Cards, containers, dividers |
| Borders (interactive) | `rgba(255,255,255, 0.15)` | Inputs, buttons |
| Borders (hover) | `rgba(255,255,255, 0.20–0.30)` | Hover states |
| Error/Red | `#f87171` | Error messages, mute-active state (keep) |
| Warning/Yellow | `#fbbf24` | Solo-active state (keep) |
| Primary button bg | `#FAFAF9` | Off-white/cream fill |

### Typography

| Element | Size | Weight | Letter-spacing | Color |
|---------|------|--------|---------------|-------|
| Hero headline | 4rem | 500 | -0.04em | `#F5F5F5` |
| Page title | 3rem | 500 | -0.03em | `#F5F5F5` |
| Section title | 2rem | 500 | -0.03em | `#F5F5F5` |
| Subtitle | 1.125rem | 400 | -0.01em | `rgba(245,245,245, 0.55)` |
| Body | 0.9375rem | 400 | -0.01em | `rgba(245,245,245, 0.55)` |
| Small | 0.875rem | 400 | -0.01em | various |
| Section label | 0.8125rem | 500 | 0.1em | `rgba(245,245,245, 0.4)` — uppercase |
| Tiny/badge | 0.75rem–0.6875rem | 400 | various | `rgba(245,245,245, 0.35)` |

**Critical:** Headlines are 500 weight, NOT 800/900. This is one of the biggest shifts from the current design.

### Borders & Shapes

- **NO border-radius anywhere.** Everything is sharp rectangles — cards, buttons, inputs, containers, badges, images, pills. Remove ALL `border-radius` values.
- Cards/containers: `1px solid rgba(255,255,255, 0.08)`, no background fill (or `#0D0D0D` max), no blur, no glassmorphism, no box-shadow.
- Hover on cards: border brightens to `rgba(255,255,255, 0.15–0.20)`.
- Only exception: slider thumb dots remain circular.

### Buttons

| Type | Background | Text | Border | Hover |
|------|-----------|------|--------|-------|
| Primary | `#FAFAF9` | black | none | `translateY(-0.5px)` |
| Ghost/secondary | transparent | `#F5F5F5` | `1px solid rgba(255,255,255, 0.15)` | border to 0.30 |

No border-radius, no gradient, no shadow on any button.

### Inputs

- Transparent background
- `1px solid rgba(255,255,255, 0.15)`
- White text
- Focus: border to `rgba(255,255,255, 0.30)`. No glow, no box-shadow on focus.
- No border-radius.

### Spacing

Extremely generous. Sections have 120–192px vertical padding. Elements breathe. Match Figma reference values: `pt-48` (192px), `pb-40` (160px), `mb-20` (80px), `mb-32` (128px).

### Component Treatments

| Component | Treatment |
|-----------|-----------|
| Nav | `#0B0B0B` bg, bottom border at white/8%, "riffd" in plain white 500 weight (not gradient), nav links in rgba 0.5, active in `#F5F5F5`. Badges = plain muted text, not colored pills. |
| Metadata badges | Bordered inline rectangles — uppercase gray label + white value, no fill, no pills |
| Tab bar | Text-only tabs, active has `#D4691F` underline at 70% opacity |
| Slider thumbs | Small circles in `#D4691F`. Track: `rgba(255,255,255, 0.1)` |
| Waveform | Thin 2px vertical bars, mostly `rgba(245,245,245, 0.3)`, every ~15th–20th bar in `#D4691F` at 70% opacity |
| Theory cards | Thin border, name in white, formula in monospace `#D4691F` at 80% opacity, description in secondary text, difficulty as plain muted text |
| Practice modules | Orange bullet dots (`#D4691F`, 0.7 opacity, 0.5rem), dark preview boxes with `background: rgba(0,0,0,0.3)` and thin border |
| Processing album art | Simple bordered square on `#0D0D0D`, NO pulsing glow, NO radial gradients |

## Implementation Order

Work through these in order. After each file, verify you haven't broken any JS by checking that all element IDs and classes referenced by JavaScript are still present.

### Step 1: `templates/base.html` — Design System Foundation

This is the most important step. Every other page inherits from this.

- Replace ALL CSS custom property values (`--bg`, `--accent`, `--text-primary`, etc.) with the Figma values listed above.
- **Remove** the entire ambient background system (the `.ambient-bg` div with gradient washes, `.wash-1`, `.wash-2`, `.wash-3`, and the radial glow overlay). Replace the page background with flat `#0B0B0B`.
- **Remove** all `border-radius` from custom properties (`--radius`, `--radius-sm`, `--radius-xs`) and set them all to `0`.
- Restyle the nav bar to match `navigation.tsx`: flat `#0B0B0B` bg, no `backdrop-filter`, plain "riffd" text in white at 500 weight (NOT gradient text), nav links in muted white, badges as plain text (not colored pills).
- Update shared `.card` class: remove glassmorphism (`backdrop-filter`, `blur`, semi-transparent bg), replace with thin `1px solid rgba(255,255,255, 0.08)` border only.
- Update all shared button styles: primary = cream `#FAFAF9` fill + black text, secondary = ghost border.
- Update all shared input styles: transparent bg, thin border, no glow on focus.
- Remove all `border-radius` from buttons, inputs, cards — everything sharp rectangles.
- Remove all `box-shadow` with colored glows (purple, teal).
- Kill `--accent-gradient`, `--accent-glow`, and all purple (`#7c5cff`) / teal (`#5eead4`) references. The only accent is `#D4691F`.

### Step 2: `templates/login.html` — Password Gate

The login page is a standalone HTML file (does NOT extend `base.html`). It has its own inline `<style>` block and must be restyled independently. Currently it uses the old purple/teal gradient scheme with glassmorphism.

**Restyle to match the new design language:**

- **Background:** Flat `#0B0B0B`. Remove the `body::before` radial gradient glow entirely.
- **Login card:** Remove glassmorphism — no `backdrop-filter: blur()`, no semi-transparent background. Use `background: #0D0D0D` or transparent. Add thin border: `1px solid rgba(255,255,255, 0.08)`. Remove `border-radius: 16px` — sharp rectangle. Remove `box-shadow`.
- **Logo "Riffd":** Remove the gradient text effect (`--accent-gradient`, `-webkit-background-clip: text`, `-webkit-text-fill-color`). Replace with plain `color: #F5F5F5`, `font-weight: 500`, `letter-spacing: -0.02em`. Lowercase "riffd".
- **Password input:** `background: transparent`. Border: `1px solid rgba(255,255,255, 0.15)`. Remove `border-radius: 10px`. Focus state: `border-color: rgba(255,255,255, 0.30)` only — remove the purple glow `box-shadow`.
- **"Enter" button:** `background: #FAFAF9`. `color: black`. `font-weight: 500`. Remove `border-radius: 10px`. Remove gradient. Remove purple `box-shadow`. Hover: `translateY(-0.5px)` only.
- **Error text:** Keep `color: #f87171`.
- **Update CSS custom properties** at the top of the style block to match the new palette — or remove them and use direct values since this file is self-contained.
- **Overall feel:** Minimal, dark, quiet. A simple card centered on a flat black background with a plain white logo, a thin-bordered input, and a cream button. No visual noise.

## Critical Rules

1. **Read every Figma reference file** before editing the corresponding template. Extract exact values — don't approximate.
2. **Never modify JavaScript.** Don't change `<script>` blocks, `onclick` handlers, element IDs, or class names that JS references (check for `getElementById`, `querySelector`, `classList` usage in the JS).
3. **Test after each file** by reviewing that all CSS changes compile correctly and no JS-referenced selectors were removed.
4. **Remove ALL border-radius** — this is the single most visible change. Every element becomes a sharp rectangle. The only exception is slider thumb dots which remain circular.
5. **Remove ALL purple (`#7c5cff`) and teal (`#5eead4`)** — replace with `#D4691F`. Exceptions: keep red (`#f87171`) for mute-active and yellow (`#fbbf24`) for solo-active on the mixer.
6. **Remove ALL glassmorphism** — no `backdrop-filter`, no `blur()`, no semi-transparent card backgrounds, no colored `box-shadow` glows.
7. **Remove the ambient animated background** entirely — the floating colored wash shapes, radial gradient overlay, all keyframe animations for washes. Background is flat `#0B0B0B`.
8. **Generous spacing** — when in doubt, add more padding between sections, not less. Match the Figma reference files.
9. **Sharp rectangles everywhere** — zero border-radius on cards, buttons, inputs, containers, images, badges, pills, chips, tabs, sliders, or any other element.
10. **`login.html` is standalone** — it does NOT inherit from `base.html`, so it must be restyled independently with its own inline CSS values.
